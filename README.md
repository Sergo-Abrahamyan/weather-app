# 🌤️ Weather App

A responsive weather dashboard that shows current weather conditions and a 5-day forecast for any city worldwide.

## 🔗 Live Demo

[View Live](https://sergo-abrahamyan.github.io/weather-app)

---

## ✨ Features

- 🌍 Search weather for any city in the world
- 🌡️ Current temperature, humidity, wind speed
- 📅 5-day weather forecast
- 🌙 Dynamic background based on weather conditions
- 📱 Fully responsive design
- 🔄 Auto-refresh every 10 minutes


## 📸 Screenshots

The app features a clean, modern UI with gradient backgrounds that change depending on the weather — sunny days show warm yellows, rainy days show cool blues, and nighttime shows dark purples.
